## Prerequisites

- PHP 8.1 - 8.3
- Node.js - v18.0.0 or newer
- MySQL

## Setup

- `composer install`
- `php artisan key:generate`
- `yarn`

## Run migrate & seeder

- `php artisan migrate`
- `php artisan db:seed`

## Serve the app

- `php artisan serve`
- `yarn dev`
- Access `localhost/login`, login and explore app
